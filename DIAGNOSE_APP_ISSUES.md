## COMMON ISSUES & QUICK FIXES

### 1. **APP WON'T START**
**Symptoms:** App crashes, black screen, or closes immediately
**Solutions:**
- Uninstall old version completely
- Clear app data and cache
- Restart phone
- Install fresh APK

### 2. **LOGIN PROBLEMS**
**Symptoms:** Can't login, "Invalid credentials" error
**Solutions:**
- Check internet connection
- Verify credentials on web version: https://mk-attendance.vercel.app
- Try different network (WiFi vs mobile data)
- Clear app data and try again

### 3. **ETHIOPIAN DATE WRONG**
**Symptoms:** Wrong month/year showing
**Status:** ‚úÖ FIXED in latest APK
**Solution:** Install the new APK with overflow fixes

### 4. **NETWORK ERRORS**
**Symptoms:** "No connection", data not loading
**Solutions:**
- Check internet connection
- Try different network
- Verify API is working: https://mk-attendance.vercel.app/api/students
- Restart app

### 5. **QR SCANNER NOT WORKING**
**Symptoms:** Camera won't open, QR codes not scanning
**Solutions:**
- Grant camera permissions in Settings
- Restart app
- Check if camera works in other apps
- Try different lighting conditions

### 6. **OVERFLOW ERRORS (YELLOW STRIPES)**
**Status:** ‚úÖ COMPLETELY FIXED in latest APK
**Solution:** Install the new APK we just built
