## ğŸ”§ **How Each Feature Works**

### **Theme Selection**
1. Tap **"Theme"** in General Settings
2. Choose from:
   - **Light Theme**: Bright interface
   - **Dark Theme**: Dark interface
   - **System**: Follows device theme

### **Auto Sync**
- **ON**: Automatically syncs data when online
- **OFF**: Manual sync only

### **Offline Mode**
- **ON**: App works without internet
- **OFF**: Requires internet connection

### **Change Password**
1. Tap **"Change Password"**
2. Enter current password
3. Enter new password
4. Confirm new password
5. Tap **"Update Password"**

### **Data Management** (Admin Only)
- **Backup**: Creates backup of all data
- **Export**: Downloads CSV/Excel files
- **Import**: Uploads student/attendance data
- **Clear Cache**: Frees up storage space
- **Restricted Access**: Only available to admin users
