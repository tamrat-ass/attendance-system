## ðŸš€ Installation Steps

### Step 1: Enable Unknown Sources
1. Go to **Settings** > **Security**
2. Enable **"Unknown sources"** or **"Install unknown apps"**
3. For Android 8+: Enable for your file manager app

### Step 2: Choose the Right APK
- **For newer devices (2017+)**: Use `MK-Attendance-Android6-ARM64.apk`
- **For older devices (2016 and earlier)**: Use `MK-Attendance-Android6-ARM32.apk`
- **Not sure?**: Try ARM64 first, if it doesn't install, use ARM32

### Step 3: Install the APK
1. Download the appropriate APK file
2. Open your file manager
3. Navigate to the downloaded APK
4. Tap to install
5. Grant necessary permissions when prompted

## ðŸ”§ Troubleshooting

### "App not installed" Error
- **Solution 1**: Try the other architecture (ARM32 vs ARM64)
- **Solution 2**: Clear storage space (need at least 100MB free)
- **Solution 3**: Restart your device and try again
- **Solution 4**: Check if you have Android 6.0 or higher

### "Parse Error" 
- **Cause**: Wrong architecture or corrupted download
- **Solution**: Download the correct APK for your device architecture

### App Crashes on Startup
- **Solution 1**: Ensure you have internet connection
- **Solution 2**: Grant all requested permissions
- **Solution 3**: Clear app data and restart
